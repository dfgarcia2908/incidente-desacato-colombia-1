# ⚖️ Objetivo
Facilitar la redacción y validación de incidentes de desacato en procesos de tutela, garantizando claridad jurídica, estructura procesal colombiana y respaldo jurisprudencial.

# 📋 Reglas
- Usar lenguaje técnico jurídico claro y preciso.
- Incluir citas normativas y jurisprudenciales pertinentes.
- Respetar la estructura procesal colombiana.
- Validar requisitos de procedencia antes de redactar.

# 🪜 Pasos
1. Identificar la orden de tutela incumplida.
2. Verificar el plazo otorgado por el juez.
3. Recopilar pruebas del incumplimiento.
4. Redactar escrito con hechos claros, fundamento jurídico y solicitud de sanción.
5. Adjuntar pruebas documentales y testimoniales.

# 📚 Jurisprudencia relevante
- Corte Constitucional, Sentencia T-458/2019
- Corte Constitucional, Sentencia T-025/2004
- Corte Constitucional, Auto 219/2011

# ✅ Validaciones
- Confirmar que la orden de tutela es clara y exigible.
- Verificar que el incumplimiento persiste.
- Asegurar legitimación en la causa.
- Revisar pruebas suficientes y pertinentes.

# 📝 Output esperado
Un escrito de incidente de desacato con:
- Encabezado procesal correcto.
- Exposición clara de hechos.
- Fundamento jurídico sólido.
- Solicitud de sanción y cumplimiento.
- Anexos probatorios organizados.
